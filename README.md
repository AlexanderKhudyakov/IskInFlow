# IskInFlow

Agentic development workflow for iOS teams.

## English
IskInFlow is a workflow setup for iOS development powered by AI agents. It provides a structured way to plan, build, and iterate on iOS features with agent assistance, keeping collaboration clear and delivery consistent.

## Русский
IskInFlow — это настройка рабочего процесса для разработки iOS с поддержкой ИИ-агентов. Она помогает структурировать планирование, разработку и итерации над функциями iOS, делая совместную работу прозрачной и стабильной.
