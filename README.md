# IskInFlow

Agentic development workflow for software teams.

## English
IskInFlow is a workflow setup for software development powered by AI agents. It provides a structured way to plan, build, and iterate on features with agent assistance, keeping collaboration clear and delivery consistent. The framework is tech-stack agnostic: project-specific paths, tools, and code-style rules live in the host project (its `CLAUDE.md` and `.claude/skills/`), never in IskInFlow itself.

## Русский
IskInFlow — это настройка рабочего процесса разработки ПО с поддержкой ИИ-агентов. Она помогает структурировать планирование, разработку и итерации над функциями, делая совместную работу прозрачной и стабильной. Фреймворк не привязан к стеку: проектно-специфичные пути, инструменты и правила код-стайла живут в host-проекте (его `CLAUDE.md` и `.claude/skills/`), а не внутри IskInFlow.
